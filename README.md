# 3D-Raycasting
Wolfenstein 3D simple remake

Run:

./cub3D map.cub

Screenshot:

./cub3D map.cub --save

Configuration file map.cub:

R - window resolution
F - floor color
C - ceiling color

WE, EA, NO, SO - textures path according side of the world

0 - empty map
1 - walls
2 - sprites
W (or N, E, S) - player on the map and his direction
